# React Tutorial

## Description
Small React practice project to learn component structure, props, and state handling.

## Tech Stack
- React
- Vite
- JavaScript

## Project Structure
src/
  server/
  client/
  styles/
    App.css
    Index.css
  App.jsx
  Index.jsx

## Project Setup (initial creation)
- Terminal: "npx create-react-app ."

## Installation
git clone <repository-url>
cd react-tutorial
npm install

## Development
npm start

## Development Server
Local: http://localhost:5173/
